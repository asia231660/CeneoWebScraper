WHAT IS THE PURPOSE OF APPLICATION?

Our application's primary goal is to gather product opinions from the Ceneo.pl website.

By providing a product code as a parameter, it extracts the product's name along with all available opinions about it.

HERE IS A DESCRIPTION OFAPPLICATION'S OPERATIONS

Welcome to our extraction page! Here, users can simply input a product code to unlock a wealth of information.

Our application swiftly retrieves the product's name and all opinions sourced from Ceneo.pl. Each opinion undergoes meticulous parsing, extracting crucial details like ID, author, recommendation, score, content, pros, cons, helpfulness ratings, publishing date, and purchase date.

Should an incorrect product code be entered, rest assured, our error prompt will guide you back on track!
